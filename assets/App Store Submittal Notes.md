### Description

> A description of your app, detailing features and functionality.

Spreadsheets are great at doing calculations, and while they can work with time, they only work with absolute time.  They're simply not designed to work with elapsed time.  With Elapsed Time Adder, you can add (and subtract) elapsed time.  

Use cases include adding split times for running or walking, keeping track of time segments in audio or video recording, and time spent on projects.

Enter the times in each row, along with an optional title for each row, and watch the total update automatically. You can type 384.6 seconds, or 74 minutes, and Elapsed Time Adder will easily work with it. If you want to subtract a row, simply hit the +/- toggle, and you'll see the row turn from green to pink.

Use the "Add Row" button to have more rows in your calculations. If you're on a Mac or an iPad with a keyboard, hitting Enter at the end of the last second cell will add a new row too.

Want to start over? Use the Reset button, and you'll be back to the default number of rows, and they'll all be empty.

When you're finished, you can export your data to a CSV suitable for opening in a spreadsheet application, or you can get it in HH:MM:SS format along with the titles for each row. This will take you to the share sheet to send your output where you desire. Note that nothing is stored in a back-end system or in the cloud; all of your data is kept on-device only.
